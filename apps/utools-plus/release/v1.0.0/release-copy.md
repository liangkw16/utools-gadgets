# uTools Plus v1.0.0 发布材料

## 版本说明

v1.0.0 是 uTools Plus 的首个公开版本，聚焦补齐 macOS 上常用但分散的系统能力。

- 新增「蓝牙」命令：查看蓝牙开关状态、读取已配对设备、搜索设备、快速连接/断开设备。
- 支持收藏常用蓝牙设备，并优先展示常用和最近操作的设备。
- 支持一键打开系统蓝牙设置，便于处理配对、权限或异常连接场景。
- 新增「扬声器 / 输出设备」命令入口，当前可快速打开系统声音设置，为后续输出设备快速切换做准备。
- 采用 macOS 专用原生 helper，插件仅面向 macOS 使用。
- 更新插件图标与发布素材。

## 插件应用介绍

uTools Plus 是一个面向 macOS 的系统能力增强插件，用来把 uTools 中暂未直接覆盖、但日常高频使用的系统操作补上。

当前版本主要提供两个入口：

1. 蓝牙管理
通过「蓝牙」「bluetooth」「Bluetooth」唤起插件后，可以查看当前蓝牙状态、已配对设备列表，并对耳机、键盘、鼠标、触控板等设备执行快速连接或断开。常用设备可以收藏，后续会固定在列表顶部，减少反复在系统设置里查找设备的操作。

2. 输出设备入口
通过「扬声器」「speaker」「输出设备」唤起插件后，可以快速打开 macOS 声音设置。这个模块已经独立预留，后续会继续扩展输出设备枚举、快速切换、收藏设备等能力。

适合场景：

- 经常在多个蓝牙耳机、键盘、鼠标、触控板之间切换。
- 希望用 uTools 命令直接处理系统设置，不想反复打开 macOS 设置面板。
- 想把蓝牙、声音、显示器、网络等系统能力集中到一个轻量工具入口。

使用说明：

- 本插件仅支持 macOS。
- 首次使用蓝牙能力时，请确保系统已完成设备配对。
- 如果连接异常，可点击「设置」进入系统蓝牙设置进行检查。
- 原生蓝牙能力依赖 macOS 系统接口，部分设备状态刷新可能受系统缓存影响，可点击「刷新」重新读取。
